# Options

| Key | Type | Default value | Description |
|-----|------|---------------|-------------|
| templateUrl | String |  | **Required** A path to your modal template. |
| controller | String |  | **Optional** A controller for your modal. |
| $scope | Object |  | **Optional** A scope that will be injected into your controller. |
| dismissEscape | Boolean | true | **Optional** A bool value determining if the modal should be able to be escapeable, will trigger a rejection.
